# make_MIUI_version_to_13_Magisk_Module 是什么？
make_MIUI_version_to_13_Magisk_Module是一个magisk 模块 是开发用来让miui+ beta 在miui12.5下 支持 应用接力 功能

# Supported Versions
理论上支持包括但不限于 MIUI 12.5 安卓11
